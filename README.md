# Seed-core
Seed: offline-first mesh financial device for the global unbanked
Seed: Offline-First Mesh Financial Device

Created by: Sophia Bratteson

Seed is a low-power, offline-first financial device designed for communities without stable internet, electricity, or banking infrastructure.

Mission

To bring financial opportunity to the 1+ billion people currently excluded from the global economy.

What This Repository Contains
	•	Prototype Code
	•	Two-laptop offline sync simulation
	•	JSON transaction schema
	•	Conflict resolution logic
	•	Architecture Diagrams
	•	Market Research & Competitor Matrix
	•	Hardware Plans (in progress)
	•	Design + Figma Screens (in progress)

Core Idea

Seed devices communicate using:
	•	LoRa mesh networking
	•	E-ink screens
	•	On-device AI
	•	Secure ledger replication without internet
